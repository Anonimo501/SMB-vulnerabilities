## SMB-vulnerabilities


Por diseño, nmap viene con varios scripts que se pueden usar para detectar varias vulnerabilidades o CVE. Específicamente para SMB.
Detecta los siguientes CVE:

2009-3103

2017-7494

ms06-025

ms07-029

ms08-067

ms10-054

ms10-061

ms17-010 (Eternal Blue)
